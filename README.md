# AQI-DDN
The present study investigated the variation of different pollutants such as PM2.5, PM10, SO2, NO2, CO and O3  over Dehradun. 
Pollution data obtained from CPCB has been analysed with meteorological variables like wind speed, direction, temperature and relative humidity. 
The analysis has been done through R programming.
